# Finance manager backend

## Installation
##### 1. Clone the project
    git clone https://github.com/lrerl/codica-test-task cloned-project
    cd cloned-project

##### 2. Install dependencies
    npm install
    
##### 3. Create .env file (see into .env.sample)

## Deploy
##### 1) From cmd
    npm run start

##### 2) From docker-compose
    docker compose up
